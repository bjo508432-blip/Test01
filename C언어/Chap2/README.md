# 가장 큰 제목 (H1)

### 세 번째로 큰 제목 (H3)
#### 네 번째로 큰 제목 (H4)
##### 다섯 번째로 큰 제목 (H5)
###### 가장 작은 제목 (H6)


# 2. 텍스트 강조 (Emphasis)
글자에 기울임, 굵게, 취소선 등의 효과를 줍니다.

*이탤릭체* : `*텍스트*` 또는 `_텍스트_`
**볼드체** : `**텍스트**` 또는 `__텍스트__`
***굵은 이탤릭체*** : `***텍스트***` 또는 `___텍스트___`
~~취소선~~ : `~~텍스트~~`


# 3. 목록 (Lists)

### 순서가 없는 목록 (*, +, - 모두 가능)
* 사과
  * 하위 목록은 들여쓰기(Tab 또는 스페이스바 2칸)를 합니다.
* 바나나
* 포도

### 순서가 있는 목록 (숫자 뒤에 마침표)
1. 첫 번째
2. 두 번째
3. 세 번째


# 4. 링크 및 이미지 (Links & Images)

### 링크 (Links)
[표시할 텍스트](연결할 URL) 구조로 작성합니다.
[구글 바로가기](https://www.google.com)

### 이미지 (Images)
링크 문법 앞에 느낌표(!)만 붙여줍니다.
![이미지 설명](이미지_링크_주소.jpg)


# 5. 인용구 (Blockquotes)
문단 맨 앞에 `>`를 붙입니다. 중첩해서 사용할 수도 있습니다.

> 이것은 인용문입니다.
>> 이것은 인용문 안의 인용문입니다.


# 6. 코드 (Code)

### 인라인 코드 (Inline Code)
문장 중간에 코드를 넣을 때는 백틱( ` ) 1개로 감싸줍니다.
이것은 `print("Hello")` 예시입니다.

### 코드 블록 (Code Block)
여러 줄의 코드를 넣을 때는 백틱 3개( ``` )로 감싸줍니다. 
시작하는 백틱 뒤에 언어 이름을 적으면 문법 강조(Syntax Highlighting)가 적용됩니다.

```python
def hello():
    print("Hello, World!")
